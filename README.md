# Kryptorious Tools

Developer workflow automation tools by **Kryptorious Quantum Biosciences, Inc.**

This is the source for the [kryptorious.gumroad.com](https://kryptorious.gumroad.com/l/jbvet) storefront and the [public tool index](https://codegero.github.io).

## What we ship
- **PyPI CLIs** — csvclean, devflow, envguard, gitsweep, dataforge, jsonguard, depguard, apiguard, draftguard, testforge, metaguard, pkgcheck
- **GitHub Actions** — python-guardian, release-forge, pr-sentinel, secret-scanner, stale-bot, docker-guardian, spell-guardian, merge-guardian, ml-guardian, dependency-bot, ai-lint, coverage-gate, go-guardian, rust-guardian, license-guardian
- **npm CLIs** — devkit, envguard, logguard, configsync, portcheck
