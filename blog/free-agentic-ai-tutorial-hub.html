We Built a Free Agentic-AI Tutorial Hub

Local-first, research-backed, and every code block actually runs.

The agentic-AI space is 90% hype. We wanted the other 10%: a curriculum that teaches you to build agents, not just chat with them — grounded in what the ecosystem actually uses.

What's in it

  • Run AI locally with Ollama — free, private inference, no API key.
  • Build your first agent from scratch — ~80 lines of plain Python, zero frameworks. You'll understand the loop.
  • Production agents with LangGraph — stateful graphs, memory, human-in-loop.
  • Multi-agent systems with AutoGen — agent teams that critique each other.
  • MCP servers — the new standard (88k★) for giving agents real tools.
  • RAG with LlamaIndex — ground agents in your own docs.
  • Prompt engineering, deployment, and agent skills.

Why local-first

Most people's first blocker is "do I have to pay / give my data to a cloud?" No. Tutorial 01 gets you free local inference in one command. Tutorial 02's agent runs entirely on your machine.

How we picked the topics

By GitHub stars, not by what's trending on Twitter. Ollama (176k★), LangChain (141k★), MCP (88k★), AutoGen (60k★), LlamaIndex (51k★) — the curriculum follows proven demand, not hype.
